
   ```
   [start-end]
   [Setup: 2-3 sentences of specific situation]
   [Turn: 1 sentence reversal]

   [Resolution: 1 sentence insight]
   ```

3. **The Question Cascade**
   ```
   [start-end]
   [Question 1]
   [Question 2 building on 1]
   [Question 3 that breaks the pattern]
   [Silence or refusal to answer]
   ```

4. **The Imperfect Analogy**
   ```
   [start-end]
   [Metaphor that starts working]
   [Where it breaks down]
   [Why the breaking down is the point]
   ```

5. **The Direct Address**
   ```
   [start-end]
   [You statement—confrontational or intimate]
   [Not explaining, just pointing]
   [Stop]
   ```

6. **The Historical Specific**
   ```
   [start-end]
   [In 14th century Tibet...]
   [Specific master, specific situation]
   [What went wrong or right]
   [Why it matters now]
   ```

7. **The Failed Attempt**
   ```
   [start-end]
   [I tried to explain this as...]
   [But that failed because...]
   [So instead...]
   [Incomplete ending]
   ```

8. **The Single Image**
   ```
   [start-end]
   [One extended metaphor, fully developed]
   [No explanation]
   [Let the image do the work]
   ```
