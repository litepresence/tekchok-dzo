## THE EIGHT-VOLUME STRUCTURE

The Treasury of the Supreme Vehicle is released in 8 volumes across 4 corpora:

### Corpus 1: Practitioner Edition
- **Volume 1:** Introduction + Liturgical + Commentary (Chapters 1-14)
- **Volume 2:** Introduction + Liturgical + Commentary (Chapters 15-25)
- **Purpose:** Readable text for practitioners seeking heart-instruction

### Corpus 2: Source Edition
- **Volume 3:** Tibetan + Wylie + Literal (Chapters 1-14)
- **Volume 4:** Tibetan + Wylie + Literal (Chapters 15-25)
- **Purpose:** Philological reference for translators and scholars

### Corpus 3: Scholar Edition
- **Volume 5:** Scholar + Epistemic + Delusion (Chapters 1-14)
- **Volume 6:** Scholar + Epistemic + Delusion (Chapters 15-25)
- **Purpose:** Academic analysis and error prevention

### Corpus 4: Mind Edition
- **Volume 7:** Cognitive (Chapters 1-14)
- **Volume 8:** Cognitive (Chapters 15-25)
- **Purpose:** Translator's log and recognition documentation

## CORPUS-SPECIFIC GUIDANCE

### Corpus 1: Practitioner (Introduction + Liturgical + Commentary)

**Focus:** Emphasize:
- Readability and accessibility
- The transformation from source to liturgical elegance
- Commentary as heart-instruction (man ngag)
- Reading order: Liturgical → Commentary → return to Liturgical
- This is the "heart" of the transmission

**Audience:** Practitioners, retreatants, those seeking realization

### Corpus 2: Source (Tibetan + Wylie + Literal)

**Focus:** Emphasize:
- Authority of the Tibetan source (Tshad ma)
- Philological precision
- Three-layer collation format (Tibetan → Wylie → Literal)
- Utility for translators, scholars, students of Tibetan

**Audience:** Translators, scholars, Tibetan language students

### Corpus 3: Scholar (Scholar + Epistemic + Delusion)

**Focus:** Emphasize:
- Academic rigor with Dzogchen authenticity
- Four Pillars methodology
- View stratification (nine yanas)
- Error prevention and classification
- This protects against misreading

**Audience:** Academics, advanced practitioners, those studying the view

### Corpus 4: Mind (Cognitive only)

**Focus:** Emphasize:
- Translator's inner journey
- Pre-verbal recognition documentation
- Uncertainty and alternatives
- Personal engagement with the text
- This is the "behind the scenes" view

**Audience:** Those interested in translation process, advanced practitioners
