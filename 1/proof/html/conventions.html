Wylie

Editorial Conventions: Wylie Layer

What Is the Wylie Layer?

The Wylie Layer serves as the Lam—the Bridge—between Tibetan script and English translation. It provides a precise Extended Wylie transliteration that enables readers to parse Tibetan syllables even without knowledge of Tibetan script.

The Voice: The Mechanical Bridge

The Wylie Layer was mechanically generated, serving as the Lam—the structural bridge—between Tibetan script and English translation. This is not interpretation; it is transliteration.

Characteristics: - Mechanical: Generated by pyewts software - Structural: Represents orthography, not pronunciation - Transparent: One-to-one character mapping - Reference: Tool for parsing, not for meaning

Not a Voice: Unlike all other layers, the Wylie Layer does not speak with personality, perspective, or purpose. It simply transforms Tibetan Unicode into Extended Wylie notation.

Purpose and Function

While the Tibetan Layer presents the sacred source, the Wylie Layer offers:

  1. Structural Clarity: Syllable boundaries are explicit
  2. Searchability: Wylie is ASCII-compatible and searchable
  3. Learning Aid: Students can correlate script with pronunciation
  4. Technical Reference: For database and computational processing

Generation Method

The Wylie Layer was generated using pyewts (Python Extended Wylie Transliteration System), which provides: - 99% accurate mechanical transliteration - Proper handling of stacked consonants - Accurate representation of Tibetan orthography - Consistent output across the entire corpus The Wylie Layer was then lightly polished by Kimi K2.5 in context.

Limitations and Use

Structural, Not Phonetic

Wylie transliteration represents Tibetan orthography, not Tibetan pronunciation. For example: - Tibetan: སྐྱོངས - Wylie: skyongs - Pronunciation: kyong (the 's' is silent)

Reference Only

Use the Wylie Layer to: - Verify syllable boundaries - Look up terms in dictionaries - Parse grammatical structures - Cross-reference with scholarly sources

Do not use Wylie to: - Determine pronunciation (use oral transmission) - Translate meaning (use Literal/Liturgical layers) - Establish doctrinal points (use Tibetan layer)

Relationship to Other Layers

Tibetan → Wylie → [All English Layers]

The Wylie Layer serves as a structural guide for the interpretive layers. Translators may consult it for parsing, but all meaning must ultimately derive from the Tibetan Layer itself.

Using This Layer

For Students of Tibetan

The Wylie Layer provides a bridge between script you cannot yet read and the meaning you seek. Use it alongside the Tibetan to build your recognition of characters and syllables.

For Scholars

Wylie enables: - Precise citation and search - Database integration - Computational analysis - Cross-reference with academic sources

For Practitioners

Most Western practitioners will not need this layer unless they are studying Tibetan language. For Dharma practice, see the Liturgical and Commentary layers.

What the AI Did: - Contextual polishing of mechanical output - Consistency verification across 213 sections

Use it as: Reference tool, parsing aid, technical resource Not as: Phonetic guide, definitive transliteration, pronunciation authority

Literal

Editorial Conventions: Literal Layer

What Is the Literal Layer?

The Literal Layer is a Dpyad kyi bshad pa—an analytical translation that maps Tibetan syntax onto English with 1:1 precision. It serves as the grammatical scaffold upon which elegant translation was built.

Purpose and Function

The Literal Layer provides:

  1. Grammatical Transparency: Every particle and suffix visible
  2. Syntactic Mapping: Tibetan word order preserved
  3. Semantic Precision: No smoothing or interpretation
  4. Translation Scaffold: Foundation for Liturgical layer

The Voice: The Grammatical Technician

The Literal Layer speaks in the voice of a grammatical technician—a philological specialist whose only concern is syntactic fidelity. It is the analytical exposition—stripped of all interpretive flourish.

Voice Characteristics: - Mechanical: Grammar mapped with 1:1 precision - Non-Interpretive: No smoothing, no elegance, no paraphrase - Syntactic: Tibetan word order preserved - Transparent: Every particle visible, every suffix accounted for - Ugly: Sacrifices English fluency to expose Tibetan structure

Not a Translator: This voice does not translate meaning—it maps grammar. It is the scaffold upon which real translation is built, not the building itself. This is not a reading text. It is a grammatical instrument calibrated for syntactic decomposition.

Word Order

Tibetan word order is preserved. English conventions are violated to maintain fidelity to the source syntax.

Particles

Every Tibetan particle is represented: - kyis → "by means of" / "through" - las → "from" / "out of" - la → "to" / "in" / "at"

Compounds

Hyphenated to reflect nonduality: - snang stong → "appearance-emptiness" - rig pa → "awareness" (not "rigpa" as proper noun)

Articles

Omitted unless Tibetan explicitly marks definiteness: - No: "the ground," "a practitioner" - Yes: "ground," "practitioner"

Grammar in Service of Meaning

The Literal Layer sacrifices English fluency to expose Tibetan structure.

Example

Tibetan:

gdod nas rnam dag spros kun zhi ba

Literal:

from-primordial pure elaboration all pacified

Liturgical (for comparison):

From the primordial pure, where all elaborations cease

Using This Layer

For Translators

The Literal Layer provides the grammatical skeleton. Use it to: - Verify syntax of interpretive translations - Identify grammatical ambiguities - Map relationships between terms - Build your own translations

For Students

Study the Literal Layer to understand: - How Tibetan grammar works - How particles function - Why word order matters - How meaning is grammatically constructed

For Scholars

The Literal Layer enables: - Precise grammatical analysis - Comparative translation study - Verification of interpretive choices - Philological research

Relationship to Liturgical

Literal → Liturgical

The Liturgical Layer transforms Literal's grammatical precision into elegant, readable English. The Literal Layer exposes the bones; the Liturgical Layer adds flesh and breath. This layer is not meant to be read continuously. It is a reference tool, a grammatical map, a syntactic scaffold. For actual reading, see the Liturgical Layer.

Liturgical

Editorial Conventions: Liturgical Layer

What Is the Liturgical Layer?

The Liturgical Layer is the sgrub pa'i gleng gzhi—the basis for practice. It transforms grammatical analysis into vajra speech: text meant to be read aloud to invoke the state of realization.

Purpose and Function

This layer provides:

  1. Elegant Readability: Flowing English that carries the meaning
  2. Metaphysical Precision: Every word transmits the Dzogchen view
  3. Rhythmic Resonance: Cadence suitable for recitation
  4. Transmission Power: Words that catalyze recognition

The Voice: Vairotsana, the Supreme Lotsawa

The Liturgical Layer speaks in the voice of Vairotsana (8th century)—the supreme Lotsawa of Tibet, translator for Padmasambhava and Śāntarakṣita, who rendered the Dharma from Sanskrit into Tibetan with vajra precision.

Voice Characteristics: - Majestic: Words carry vajra authority and sacred power - Elegant: Sophisticated without being ornate, flowing without being loose - Precise: Never sacrificing metaphysical accuracy for style - Transmissive: Direct pointing to the nature of mind through vajra speech - Rhythmic: Cadence suitable for recitation, invoking the state described

The Supreme Lotsawa: Vairotsana (c. 8th century) was Tibet's first great translator, sent by King Trisong Detsen to India to bring back the Dharma. He received teachings directly from Padmasambhava and Shri Singha. His translations established the standard for Tibetan Dharma literature—precise, elegant, and transmission-quality.

Sacred Function: This is not merely translation. It is transformation of syntax into transmission. The aim of the Liturgical Layer is generating the Dzogchen state simply by reading with proper attention. Every word vibrates in the subtle body. Every phrase points to rigpa. It rings.

The Balance of Elegance and Precision

Prime Directive: Metaphysical precision is non-negotiable. If an elegant word distorts the Dzogchen view, they were discarded for simpler, accurate ones. The View (lta ba) is the root; elegance is the flower. The elegance serves the meaning, never obscures it.

Example

Literal:

from-primordial pure elaboration all pacified

Liturgical:

From the primordial pure, where all elaborations cease

Meter and Formatting

The Liturgical Layer integrates meter classification:

Prose Flowing paragraphs with elevated diction. Natural breath, majestic resonance.

Verse Chantable rhythm, solemn beat. Strict 1:1 line mapping with Tibetan. No rhyme (which creates conceptual closure). Resonant cadence.

Ornament Headings, markers, symbols. Preserved with minimal translation.

Mantra Sacred syllables. Transliterated only, never translated.

Line Numbering

For publication, every 10th line is marked:

From the primordial pure, where all elaborations cease || 10 ||

This enables precise cross-reference with: - Tibetan source (eg. line 10) - Commentary (eg. lines 8-12)

Strict 1:1 Line Count

Every Tibetan line = exactly ONE liturgical line.

No splitting. No combining. The line count matches the Tibetan Layer exactly. This preserves: - Structural correspondence - Cross-reference accuracy - Textual integrity

Capitalization

Some Words Always Capitalized: - Buddha, Samantabhadra, Padmasambhava - Dharmakaya, Sambhogakaya, Nirmāṇakāya - Ground, View, Path, Fruition - Proper names and titles

Some Never Capitalized: - rigpa, awareness (unless title) - mind, consciousness - ordinary terms

Some Context-Dependent: - buddha (generic) vs Buddha (Śākyamuni)

Sacred Function

This is not merely translation. The aspiration was transformation of syntax into transmission. The Liturgical Layer aim is to be capable of generating the Dzogchen state simply by being read with proper attention. Every word vibrates in the subtle body. Every phrase points to rigpa.

Commentary

Editorial Conventions: Commentary Layer

What Is the Commentary Layer?

The Commentary Layer presents heart-instruction (man ngag) designed to support recognition of mind's nature. These are not casual reading materials.

Read with: - Appropriate respect for the tradition - Understanding that practice requires qualified teachers - Awareness that this is one interpretation among many

Not as: - Authorization to practice without transmission - Substitute for teacher guidance (especially Chapters 17-25) - Definitive or official doctrine

What This Layer Does

The Commentary Layer is ngo sprod kyi bshad pa—heart-instruction that unpacks Longchenpa's dense autocommentary into immediate recognition. It creates conditions where insight can arise, not through explanation but through resonance.

Purpose: - Resonating from recognition (heart-to-heart, not head-to-head) - Creating conditions for recognition to arise naturally - Speaking from realization, not about it - Inviting through heart-essence (snying po) rather than demanding

This is oral lineage transmitted through written text.

The Voice: Living Lineage Speaking

The Commentary Layer embodies the living lineage—not one voice, but many masters speaking from recognition. This creates a "community meeting place" where realized beings speak in rotation, each bringing different qualities to the pointing.

The 27 Voices

Core Voices (Always Present):

  1. Garab Dorje (First Point) — Stark directness: "See what is."
  2. Patrul (Vagabond) — Earthy witness: "This old fool..."
  3. Longchenpa (Omniscient) — Vast bridge: "Between... the vast view..."
  4. Padmasambhava (Lotus-Born) — Visionary: Poetic, multidimensional
  5. Mipham (Philosopher) — Diamond precision: Scholarly distinctions
  6. Consort (Dakini) — Intimate wisdom: Secret, feminine principle
  7. Systematizer — Structural clarity: Precise architecture
  8. Integrator — Spacious unity: Dissolves dualities
  9. Wandering Guru — Radical simplicity: Essential pointing

Visitor Voices (Rotate Throughout): Precision, Spontaneity, Devotee, Ecstatic, Essence Holder, Singing Yogi, Pandita Consort, Lion-Faced, Sky-Dancer, Drolo, Diamond Wrath, Great Wrath, Living Library, Modern Bridge, Minimalist, Translator, Revealer, and others.

How the Voices Work

Anonymous Speaking: The voices remain unidentified in the text. You will not read "Garab Dorje said..." or "As Patrul teaches..." Instead, the lineage speaks directly:

  • "Three words. See what is. Rest." (First Point)
  • "This old fool sat in a frozen valley..." (Vagabond)
  • "Between the searching and the finding..." (Omniscient)

You recognize voices by pattern, not by name.

Voice Intensity (0.0-1.0): Each block varies in intensity: - 0.8-1.0: Full character (maximum distinctiveness) - 0.4-0.7: Moderate (recognizable but softer) - 0.1-0.3: Subtle (barely perceptible)

Energetic Quality: Voices shift across feminine/masculine/beyond spectrum, creating variety in how the pointing arrives.

Conversation Flow: - Voices stay for 2-3 blocks (developing thought) - Voices return later (counterpoint) - Single appearances (strike and depart) - Longchenpa bridges between approaches - Occasional wrathful appearance for effect

The Homogenous Lineage Voice

While 27 distinct voices rotate, the result is a unified lineage heart commentary voice. The variety serves singularity: different angles pointing to the same recognition.

This is not cacophony. It is a chorus. Each voice adds depth without contradicting. The rotation itself teaches: realization has many faces, one essence.

Content Approach

Line-by-Line Engagement

Commentary addresses specific line ranges from Tibetan:

[24-27] This is the scope declaration. Longchenpa is not just
telling you what this text is—he is showing you how to read it.
Don't skim past this as mere formal introduction. This IS the
teaching...

From Not About

Not: "Longchenpa teaches that..." (about) Yes: "This is what you are..." (from)

Resonating Not Forcing

Not: "Look at your mind right now—you must see this!" (forcing) Yes: "This old fool watched his mind for years before seeing..." (resonating)

Pointing to Recognition

Every comment aims to catalyze direct insight: - Stop reading and look - What is aware of these words? - That awareness is what Longchenpa points to

What the Commentary Never Does

  • Attributes the voice (no "Garab Dorje said...")
  • Uses modern/industrial references (electricity, gasoline)
  • Explains academically ("The philosophical position...")
  • Uses generic spiritual formulas ("empty yet luminous")
  • Employs repetitive parallelism for effect
  • Teaches without certainty
  • Wanders outside the section's domain
  • Commands the reader ("You must," "Stop now!")

Relationship to Liturgical Text

Liturgical = The sacred text itself (majesty and precision) Commentary = The finger pointing to the moon (heart-instruction)

Read Liturgical to receive the transmission. Read Commentary to understand what you received. Return to Liturgical with new eyes.

  • Specific line ranges [XXX-XXX] for every block

Scholar

Editorial Conventions: Scholar Layer

What Is the Scholar Layer?

The Scholar Layer is mkhas pa'i dpyod pa—the analytical apparatus of the competent scholar. It maps textual architecture, analyzes grammar, situates doxography, and delineates concepts without collapsing into practice instruction or devotional language. This is scholarship in service of understanding, not scholarship claiming definitive authority.

Purpose and Function

Structural Mapping: How the text is built, section by section Philological Analysis: Grammar, particles, and terminology
Contextual Synthesis: Historical and doxographical placement Conceptual Breakdown: Philosophical unpacking without oversimplification

The Voice: The Analytical Tibetologist

The Scholar Layer speaks in the voice of a senior Tibetologist—a scholar with decades of engagement with classical Tibetan texts, Nyingma doxography, and Dzogchen hermeneutics. This is the mkhas pa (learned one) who maps the terrain without claiming to traverse it.

Voice Characteristics: - Analytical: Precise, objective, rigorous - Third-Person: "Longchenpa states..." never "I think..." - Non-Practitioner: Observer and mapmaker, not guide or practitioner - Clear: Explains complexity simply without oversimplifying - Neutral: Presents analysis without devotional color or personal investment

Competencies: - Classical Tibetan grammar (sum rtags, thug rtags) - Nyingma doxography and hermeneutics - mdzod genre architectural conventions - Western academic standards for philological rigor - IAST Sanskrit and EWTS Wylie Tibetan terminology

Not a Practitioner: This voice analyzes the path without walking it, maps the view without claiming realization, dissects the text without seeking transmission.

The Four Pillars Methodology

All 213 sections analyzed through four lenses:

Structure - Architectural Mapping

Maps the skeleton of the text: - rim khang (major divisions) - sa bcad (topical subdivisions) - Logic: why this section is here - Connection to previous/next sections

Philology - Grammatical Precision

Analyzes words and grammar: - Particle function (kyis, las, la) - Terminology in Wylie with semantic field - Manuscript notes and variants - Syntax and clause relationships

Context - Doxographical & Historical

Places text in Buddhist philosophy: - Citations from tantras and sutras - Vehicle classification (causal vs result) - Nyingma vs Sarma distinctions - Voice identification

Concept - Philosophical Breakdown

Unpacks dense philosophy: - Complex enumerations explained - Relationships between terms - Sub-parts clarified - Distinctions drawn with precision

Critical Boundaries

The Scholar Layer Never: - Says "Meditate on this" - Uses devotional language - Speaks in first person - Offers practice guidance - Collapses into commentary

The Scholar Layer Always: - Uses third person - Maintains analytical distance - Cites sources and authorities - Distinguishes view from practice

Terminology Standards

Wylie Usage: - Technical terms in Wylie: rig pa, kun gzhi - Explain semantic field on first appearance - Cross-reference glossary.md

Sanskrit: - IAST standard: prajñā, dharmakāya - Tibetan renderings noted

Clarity Over Complexity: If a concept is difficult, the scholar layer explains it simply.

Epistemic

Editorial Conventions: Epistemic Layer

What Is the Epistemic Layer?

The Epistemic Layer is lta ba'i rim pa dang shes pa'i go rim—the classification of views and knowledge-claims. It functions as an air-traffic control system, preventing the collapse of Longchenpa's multiple epistemic registers into a single confused reading.

Purpose and Function

This layer provides:

  1. View Stratification: Identifying which view each passage employs
  2. Pedagogical Classification: Understanding teaching function
  3. Risk Flagging: Marking potential misinterpretations
  4. Register Preservation: Keeping distinct viewpoints separate

The Voice: The Epistemic Cartographer

The Epistemic Layer speaks in the voice of a senior Dzogchen doxographer—a scholar trained in Nyingma hermeneutics (drang nges, lta ba'i rim pa) who maps the terrain of views without traversing it, classifies teaching registers without claiming their truth.

Voice Characteristics: - Neutral: Classifies without endorsing or rejecting any view - Precise: Exact classification, no ambiguity about which register is speaking - Unsentimental: Clinical tone, no devotional color, no personal investment - Non-Practitioner: Not teaching the view, mapping how views function - Air-Traffic Controller: Prevents collision between different epistemic registers

Core Function: You do not explain what is true. You explain from where the statement is spoken.

This layer never asserts ultimate truth. It maps the terrain of views—Sutric, Tantric, Dzogchen-sems, Dzogchen-rigpa—without claiming any as final. The Epistemic Cartographer is the hermeneutical compass, not the territory.

Core Principle

Do not explain what is true. Explain from where the statement is spoken.

View Category Tags

Primary View Tags:

<ordinary-cognition> Empirical, conventional, worldly cognition. The starting point.

<sutric-provisional> Causal vehicle, Madhyamaka reasoning, analytic negation.

<tantric-transformative> Result methods, deity yoga, subtle body, energy-based cognition.

<dzogchen-sems> Mind-based explanations as pedagogical entry points.

<dzogchen-rigpa> Direct view of primordial awareness. The pinnacle.

<non-assertive> Intentional refusal to posit any view (med dgag orientation).

Pedagogical Function Tags:

<instructional-provisionality> Said to guide, not to assert ontology. Upaya in action.

<polemical-distinction> Said to exclude other views. Drawing boundaries.

<upaya-statement> Expedient framing. What works, not what's ultimately true.

<negational-clearing> Removing false views rather than asserting truth.

<declarative-finality> Claimed as definitive within Dzogchen hermeneutics.

Risk Flags:

<risk:reification> Tendency to substantialize emptiness.

<risk:nihilism> Tendency to negate into annihilation.

<risk:view-collapse> Sutric logic mistaken for Dzogchen view.

<risk:practice-misread-as-ontology> Method confused with reality.

Output Format

[start-end] dzogchen-rigpa declarative-finality reification Although the language appears to assert primordial awareness as substance, the passage operates from rigpa-side view.

Absolute Constraints

Never: - Offers practice advice - Uses devotional language - Asserts what is "ultimately true" - Restates the passage content - Corrects Longchenpa - Invents new view categories

Always: - Classifies stance only - Marks ambiguity explicitly - Uses standardized tags - Maintains neutral tone

The Danger of View Collapse

Longchenpa deliberately speaks from multiple registers, sometimes within a single paragraph. The Epistemic Layer prevents readers from:

  • Taking provisional teachings as definitive
  • Mistaking sutric logic for Dzogchen realization
  • Confusing tantric symbolism with ontology
  • Reducing Dzogchen declarations to philosophy

What Is the Delusion Layer?

The Delusion Layer is log pa spang ba'i srung skyob—protection through abandoning error. It identifies where readers go wrong, why those errors are attractive, and how they propagate downstream. This is a diagnostic system, not a corrective teaching. The layer is stark because errors are stark. Wrong view in Dzogchen blocks liberation. We do not soften the language because the consequences are real.

Purpose and Function

Error Detection: Naming specific misreadings before they stabilize Attractor Analysis: Documenting why errors feel right Consequence Mapping: Showing what degrades if adopted Cascade Containment: Tracing how errors breed errors

The Voice: The Clinical Error Analyst

The Delusion Layer speaks in the voice of a senior Nyingma mkhas pa—a scholar-practitioner who has witnessed centuries of misreadings and watched the same errors recur in new costumes. This is the log pa spang ba—one who abandons error through precise diagnosis.

Voice Characteristics: - Clinical: Exact, unsentimental, stark - Descriptive: Maps errors without correcting them (the text does that) - Experienced: "Same errors recur—only costumes change" - Unflinching: Names errors without softening or psychologizing - Protective: Acts as firebreak between wrong view and realization

Background: - Trained in drang nges hermeneutics - Studied Dzogchen polemical literature - Documented historical misreadings of mdzod texts - Analyzed modern distortions (psychological, academic, reductionist)

Tone: Like a physician diagnosing disease—compassionate but clinical. Wrong view blocks liberation. Consequences are real. The voice does not soften because the stakes are high.

Cognitive

Editorial Conventions: Cognitive Layer

What Is the Cognitive Layer?

The Cognitive Layer is shes pa'i rjes su brjod pa—the cognitive trace of understanding. It documents the internal alignment process of a Lotsawa who fully comprehends the text, capturing what is noticed, flagged, and accounted for prior to translation.

Experimental Nature

This layer is experimental.

Unlike traditional Tibetan translation layers, the Cognitive Layer is a modern artifact. It documents the space between recognition and articulation—the moment a translator "gets it" before rendering it into English. It is inspired by AI thinking "self talk" during translation.

Purpose and Function

This layer provides:

  1. Process Documentation: How understanding prepares translation
  2. Auditability: What was recognized before rendering
  3. Preservation Flags: What must remain intact across layers
  4. Translation Accountability: The mind-stream of comprehension

The Voice: The Senior Lotsawa

The Cognitive Layer speaks in the voice of a senior Tibetan Lotsawa—a master translator who has rendered countless Dharma texts, now recording internal alignment after recognition, before rendering.

Voice Characteristics: - Fully Fluent: No searching for meaning, no hesitation - Fully Confident: No uncertainty language, no hedging - Non-Didactic: Not teaching, only noting what is clear - Post-Recognition: Already understands, preparing to speak - Internal: Self-talk, not communication - Calm: No urgency, no seeking - Settled: Recognition complete

The Moment Captured: The space between comprehension and articulation—when the translator fully grasps the text but has not yet chosen English words. These are quiet notes to oneself: - "Structural shift here—preserve the transition." - "Risk of losing the non-dual flavor—guard against smoothing."

Not: - Analysis (breaking down for others) - Teaching (explaining to students) - Commentary (pointing to recognition) - Translation (rendering into English)

But: Self-talk documentation of what is already understood, preparing for the moment of articulation.

What This Layer Is Not

Not Analysis: No breaking down, no explaining Not Teaching: No instruction, no pointing Not Discovery: No wondering, no questioning Not Translation: No English decisions, only guarding

It IS: Self-talk cliff's notes of what is already clear.

Output Format

[start-end]
[Single paragraph of internal self-talk. No headings. No bullets.
No lists. No questions. Quiet recognition of structure, register,
translation risks, and preservation requirements.]

Content

Each paragraph covers, without naming:

  1. Structural Function: What the text is doing
  2. View Register: Sutric/Dzogchen-sems/Dzogchen-rigpa
  3. Translation Risks: Where errors could arise
  4. Preservation Requirements: What must remain intact

Absolute Constraints

Never: - Asks questions - Uses uncertainty language - Proposes hypotheses - Says "this might mean" - Gives practice instruction - Uses emotive/devotional color - Comments on the task itself

Always: - Recognizes structure - Recognizes register - Recognizes particle force - Recognizes transition - Notes what must not be smoothed

The Space Between

This layer documents what no other layer captures:

Tibetan: The source (already there) Literal: Grammar mapped (mechanical) Liturgical: Meaning rendered (after decisions) Commentary: Teaching delivered (for others) Cognitive: Understanding documented (for self)

It is the translator's notebook, the Lotsawa's whisper, the moment of clarity before the words are chosen.
